# Vsche(Vスケ)
![logo](https://user-images.githubusercontent.com/50011163/113076347-cc025d00-9209-11eb-8249-a65e9cbc1884.png)<br>
Vtuberの配信予定をLINEで簡単チェック！<br>
ホロライブVtuberの名前や愛称を入力すると、今日の配信予定を返してくれるLINEBOTです<br>
